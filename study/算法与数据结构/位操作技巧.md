# 大小写字母转换
- 或空格可以得到小写字母  'A' | ' ' = 'a' (a结果也是a)
- 与下划线可以得到大写字母 'a' & '_' = 'A'(A结果还是A)
- 异或空格可以得到字母大小写转换  'A' ^ ' ' = 'a' 而 'a' ^ ' ' = 'A'
(恰巧这些字符对应的数字通过位运算就能得到正确的结果)

# 判断异号

(-1 ^ 1)<0 = true
(1^2)>0 = true

如果用乘积可能会溢出（补码编码）

# 交换两个数
a = a ^ b
b = b ^ a
a = a ^ b(异或回来)

# n&(n-1)
消除最后一个1（不需要1一定是在第一位）
判断是否为2的指数 就可以n&n-1 因为2的指数一定只有一个二进制1 ，消除了就等于0
